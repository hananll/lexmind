# LexMind

AI-powered language learning platform that helps learners improve vocabulary, reading, writing and speaking through personalized AI feedback.

## Features

- AI Vocabulary Assistant
- Reading Analysis
- AI Chat Tutor
- Smart Review System
- Personalized Learning Analytics

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Gemini API
- Docker
